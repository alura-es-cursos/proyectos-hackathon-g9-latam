Contexto

Sobre el Hackathon

Una maratón de innovación para los alumnos del programa ONE, donde ponen en práctica los conocimientos adquiridos, desarrollando soluciones reales con tecnología e inteligencia artificial.

🎯

Objetivo

Desarrollar MVPs funcionales que integren Data Science, API REST y servicios de nube OCI, solucionando problemas reales del mercado.

👥

Equipos

Los alumnos se organizan en grupos multidisciplinares en la plataforma NoCountry, uniendo perfiles de Back-End, Ciencia de Datos y más.

☁️

Tecnologías

Python, Scikit-Learn, Java, Spring Boot y Oracle Cloud Infrastructure (OCI) son las principales tecnologías involucradas en nuestros proyectos.

🏆

Entregas

Cada equipo entrega un notebook de Data Science, una API REST documentada y integración con al menos un servicio OCI.

NLP · Gestión del conocimiento

TechMind

Classificación e Organización Inteligente de Conteúdos

Descripción del Proyecto

Crear una solución inteligente capaz de recibir contenidos técnicos en formato textual y procesarlos automáticamente, clasificándolos en categorías relevantes, extrayéndolos palabras clave y enriqueciendo la información con metadatos generados por el modelo.

El objetivo es facilitar la organización y el acceso a grandes volúmenes de información, haciendo repositorios de conocimiento más eficientes y navegables.

🏢 Sector

Plataformas educativas, documentación técnica y gestión del conocimiento corporativo.

Stack Técnica

Python Pandas Scikit-Learn TF-IDF Java Spring Boot OCI

📦 Entregas Obligatorias (MVP)

  • Notebook con EDA, entrenamiento y serialización del modelo
  • API REST con endpoint POST /contenido
  • Retorno JSON con categoría, probabilidad y palabras clave
  • Integración con al menos un servicio OCI
  • Mínimo de 3 ejemplos de uso documentados

⚡ Ejemplo de Requisición

{
                "titulo": "Introducción a Spring Boot",
                "texto": "Conceptos básicos para
                    la creación de APIs REST con
                    Java y Spring Boot."
                  }

✅ Ejemplo de Respuesta

{
  "categoria": "Backend",
  "probabilidad": 0.89,
  "informaciones_adicionales": [
    "Java", "Spring Boot",
    "API REST"
  ]
}

🚀 Recursos Opcionales

  • Recomendación de contenidos relacionados
  • Búsqueda semántica
  • Panel de control simple
  • Procesamiento por lotes (CSV)
  • Contenedorización con Docker

Fintech · Educación Financiera

FinanceAI

Asistente Inteligente de Salud Financiera

Descripción del Proyecto

Crear una solución inteligente capaz de analizar el comportamiento financeiro de un usuario a partir de sus transacciones, generando una visión más completa de su salud financeira.

La solución clasifica gastos automáticamente, identifica patrones de consumo, define el perfil financiero del usuario y presenta recomendaciones personalizadas para mejorar su organización financiera.

🏢 Sector

Fintech, bancos digitales, carteras digitales y plataformas de educación financiera.

Stack Técnica

Python Pandas Scikit-Learn Java Spring Boot OCI

📦 Entregas Obligatorias (MVP)

  • Notebook con EDA, ingeniería de atributos y modelos
  • API REST con endpoints de clasificación de gastos
  • Clasificación de perfil financiero (ej: Organizado, Endeudado)
  • Recomendaciones personalizadas en JSON
  • Alertas de gastos elevados
  • Integración con al menos un servicio OCI

⚡ Ejemplo de Solicitud

{
  "descripcion": "Uber",
  "valor": 45.00,
  "renta_mensual": 3500,
  "frecuencia_ahorro": "baja",
  "nivel_endeudamiento": "moderado"
}

✅ Ejemplo de Respuesta

{
  "categoria_gasto": "Transporte",
  "perfil_financiero": "Moderado",
  "recomendaciones": [
    "Considerar alternativas de
    transporte más económico",
    "Establecer un presupuesto
    mensual transporte"
  ]
}

🚀 Recursos Opcionales

  • Panel (Dashboard) de seguimiento
  • Historial de análisis
  • Exportación de informes
  • Procesamiento por lotes vía CSV
  • Contenedorización con Docker

⚡
Sostenibilidad · Energía

EnergiAI

Inteligencia para el Consumo de Energía

Descripción del Proyecto

Crear una solución inteligente capaz de analizar patrones de consumo de energía eléctrica y generar información que auxilie en la toma de decisiones relacionadas con la eficiencia energética.

La aplicación utiliza Ciencia de Datos para identificar patrones y clasificar el perfil energético en categorías como Eficiente, Moderado o Ineficiente, proporcionando recomendaciones y estimaciones de impacto financiero (tarifa de referencia: R$ 0,75/kWh).

🏢 Sector

Sostenibilidad, energía y hogares inteligentes - residencias y pequeños establecimientos.

Stack Técnica

Python Scikit-Learn Random Forest Java Spring Boot OCI

📦 Entregas Obligatorias (MVP)

  • Notebook con EDA y modelo supervisado entrenado
  • Endpoint POST /analise-energetica
  • Clasificación de perfil energético con probabilidad
  • Recomendaciones de optimización energética
  • Estimación de costo mensual (R$/mes)
  • Integración con al menos un servicio OCI

⚡ Ejemplo de Solicitud

{
  "consumo_kwh": 420,
  "uso_horario_pico": true,
  "cantidad_equipos": 10,
  "tipo_inmueble": "Casa",
  "horas_alto_consumo": 8
}

✅ Ejemplo de Respuesta

{
  "categoria": "Ineficiente",
  "probabilidad": 0.81,
  "costo_estimado_mensual": 315.00,
  "recomendaciones": [
    "Reducir uso en horarios
    de pico",
    "Evaluar equipos de
    alto consumo"
  ]
}

🚀 Recursos Opcionales

  • Panel (Dashboard) de seguimiento
  • Historial de análisis
  • Comparación entre períodos
  • Simulación de escenarios de ahorro
  • Ranking de eficiencia energética
OCI sugerido: Object Storage · Compute · Functions

Tecnología

Rutas Técnicas

Cada equipo está compuesto por estudiantes de diferentes rutas, colaborando para entregar una solución completa.

🔬

Ciencia de Datos

Exploración, limpieza y modelado de datos con Python, Pandas y Scikit-Learn. Entrega de notebook con modelo entrenado y serializado.

  • EDA y preprocesamiento
  • Ingeniería de atributos
  • Entrenamiento y evaluación
  • Serialización del modelo
⚙️

Back-End

Desarrollo de API REST con Java y Spring Boot, integrando el modelo de Ciencia de Datos y exponiendo endpoints documentados.

  • API REST con Spring Boot
  • Integración con modelo DS
  • Validación y tratamiento de errores
  • Documentación de los endpoints
☁️

Oracle Cloud (OCI)

Integración obligatoria con al menos un servicio OCI para almacenamiento, hospedaje o procesamiento de la solución.

  • Object Storage
  • OCI Compute
  • OCI Functions
  • Banco de datos (opcional)
